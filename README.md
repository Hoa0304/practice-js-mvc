## HOW TO RUN

- Dillinger requires Node.js v18+ to run.
- Install the dependencies and devDependencies and start the server.

```
cd practice-js-mvc
npm i
npm run watch

http://localhost:8000
```

```
cd practice-js-mvc
$npm i
$npm run dev

Open on web: http://localhost:1234
```
